# Dominance Prototype (Auto Pong)

A mesmerizing, automated territory conquest game where two entities compete to "paint" the board in their color.

## Gameplay
- **Territory Capture**: The board is shared between Player A and Player B.
- **Automated Combatants**: Two balls move across the screen, capturing opponent tiles on impact.
- **Physics**: Balls bounce with a slight random jitter to prevent infinite loops.

## Controls
- **Space**: Pause/Resume
- **R**: Reset the game

## Tech
- Vanilla JavaScript
- HTML5 Canvas
